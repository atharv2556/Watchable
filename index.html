The Legend of
Jetha Lal & Watchable

Founded in the heart of Bardoli, Gujarat, in 1992, Watchable was born from Jetha Lal's personal obsession with timing movements and restoration. Over the past 30 years, our workshop has restored thousands of classic family heirlooms, bringing precise mechanical heritage back to life.

What started as a tiny workbench has grown into a prestigious lounge welcoming over 55 daily recurring patrons. Every watch we sell, repair, or rent goes through Jetha Lal's personal signature quality inspections.

"A watch doesn't just tell time. It tells your story. Our mission is to keep your stories ticking for generations." — Jetha Lal, Founder & Head Horologist

Google OAuth & Supabase Setup Guide

  1. Setup Google Cloud Console:
    • Navigate to the Google Cloud Console and create a new project.
    • Configure the OAuth Consent Screen (set user type to External, fill app details).
    • Go to Credentials -> Create Credentials -> OAuth Client ID.
    • Select Application Type: Web Application.
    • Add Authorized JavaScript Origins: http://localhost:5500 (or your current web origin).
    • Add Authorized Redirect URIs: https://sebusfrzbejgjnlmsfvv.supabase.co/auth/v1/callback.
    • Click Create to retrieve your Client ID and Client Secret.
  2. Setup Supabase Dashboard:
    • Open your project at Supabase Dashboard.
    • Navigate to Authentication -> Providers -> Google.
    • Enable the Google provider.
    • Copy the Client ID and Client Secret from Google Cloud Console and paste them here.
    • Save the provider configurations.
  3. Enable Client-side Integration:
    • Paste the Client ID and Client Secret in the Developer Config tab of this website.
    • Set a custom AES Encryption Key to protect user database rows.
    • Save the settings, then sign in using the Google button!
